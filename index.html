Take a look at some of my projects

Sudoku Solver

I wrote the logic behind a sudoku solver in JavaScript then wrote post logic for solving and checking answers on a Node/Express application server. You can input your own board state and have the options to either solve the board based on your prompt, or check values for specific input and receive instant feedback on if it is a valid answer.

Sample Survey

This was a project I did early on to practice HTML and CSS. It is a simple sample survey using form, radio buttons, text area, and drop down menus.

Tree-map Data Visualization

I was given three JSON object links and tasked with visualizing one of them by making a tree-map scaled to the values of the data in the objects. I created a dynamic graph that you can interact with by hovering your mouse over entries to receive more information. I also made it so that you could toggle between the three different JSON objects to look at maps of each JSON Object and the tree-map would dynamically change to fit the new data.

Technical Information Page

This was a project designed to practice HTML and CSS elements. It is a single page with anchored links to the different title elements of the page.

URL Shortener Microservice

Using Node and Express with MongoDB, I wrote code that posts user input URLs into a database, and returns a shortened URL back to the user that they can then use to go to the original URL location. I accounted for redundancy and made sure that if a duplicate URL was entered by a user, they would receive the already existing database entry rather than creating another entry to keep the database organized.

American-British Translator

Using several objects of key value pairs that show an American spelling or version of a word or phrase and then its British counterpart, I wrote the logic to translate a user input string into the opposite version. The starting key value pairs ranged from one word phrases to multiple word phrases, so I had to account for that when searching for matching phrases from the database to translate.

Drum Machine in React

For this project I was given nine links that each had a sound element and told to make an interactive drum machine that would play back each sound when clicked or the corresponding key was pushed. I build it in React JavaScript using multiple different states, useEffects, and useRef. When playing back each sound, the name of the sound will be displayed for the user to see.

JavaScript Calculator in React

I made a JavaScript Calculator in React that can do all basic calculator math functions. If you chain math functions, it will keep the working calculation of the previous entry and solve based on that. You can accurately use decimals and negative numbers, and even change what math function you had entered if you clicked the wrong one by mistake.

Choropleth Map of USA Education Data

Using the visualization language D3 in JavaScript, I used a linked JSON object to create an interactive map of the United States that showed the education density of each state.
